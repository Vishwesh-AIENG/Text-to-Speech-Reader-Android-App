## вҡЎ Performance Highlights

- рҹ”’ **Background isolation** вҖ” exiting any mode immediately unbinds the camera, stops TTS/STT and cancels all coroutines
- рҹ§  **`derivedStateOf`** вҖ” waveform and UI state changes only trigger recomposition for the composables that actually changed
- рҹ“Ұ **LRU caches** вҖ” translation results and PDF page bitmaps are cached to prevent redundant processing
- рҹҺЁ **Canvas allocations hoisted** вҖ” `Brush`, `Path`, and `AnnotatedString` objects are `remember`-ed to avoid per-frame allocations
- рҹ–јпёҸ **VBO dirty flag** вҖ” OpenGL buffers only re-upload geometry when the word list actually changes
- рҹ”‘ **Unique utterance IDs** вҖ” `nanoTime()`-based IDs prevent TTS completion callbacks from firing on stale utterances

## рҹҡҖ Getting Started

### Prerequisites
- Android Studio Hedgehog or later
- Android device or emulator with API 24+
- ~1.3 GB free storage for the on-device Gemma AI model (downloaded in-app on first use)

### Build & Run

```bash
# Clone the repository
git clone https://github.com/Vishwesh-AIENG/Text-to-Speech-Reader-Android-App.git
cd Text-to-Speech-Reader-Android-App

# Debug build
./gradlew assembleDebug

# Release AAB (requires keystore.properties вҖ” see below)
./gradlew bundleRelease

# Run tests
./gradlew test

# Lint
./gradlew lintRelease
```

### Release Signing

Create `keystore.properties` at the project root (already git-ignored):

```properties
storeFile=/path/to/your/upload.jks
storePassword=your_store_password
keyAlias=your_key_alias
keyPassword=your_key_password
```
